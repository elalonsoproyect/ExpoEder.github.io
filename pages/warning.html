Cause and effect graph in black box tests:

The cause-effect graph is included in the black box testing technique that underlines the relationship between a given result and all the factors that affect the result. It is used to write dynamic test cases.

Dynamic test cases are used when your code works dynamically based on user input. For example, while using an email account, when you enter a valid email, the system accepts it, but when you enter an invalid email, it throws an error message. In this technique, input conditions are assigned with causes and the result of these input conditions with effects.
